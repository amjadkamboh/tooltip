== Description ==

== A simple and easy way to add WordPress Tooltips to display informative text when users hover over an element in Gutenberg and the classic editor.== 

With this plugin you add tooltips to your wordpress site. Tooltips will show when target element in Gutenberg and the classic editor is hovered over. On mobile devices tooltips show when target element is tapped.

== Installation ==

This section describes how to install the plugin and get it working.

== 1) INSTALL == 
Go to the WordPress Dashboard “Add New Plugin” section.
Search For WordPress Tooltip - Shortcodes & Gutenberg.
Install, then Activate it.
OR:
Unzip (if it is zipped) and Upload WordPress Tooltip - Shortcodes & Gutenberg folder to the /wp-content/plugins/ directory
Activate the plugin through the ‘Plugins’ menu in WordPress

e.g.

1. Upload `WordPress Tooltip - Shortcodes & Gutenberg` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
